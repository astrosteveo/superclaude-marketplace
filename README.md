# SuperClaude Marketplace

Cherry-picked, battle-tested plugins for Claude Code, designed, developed, and curated by [Steven Mosley](https://github.com/astrosteveo). This marketplace focuses on core development and debugging skills, leveraging Agent Teams, Agents, Skills, Commands, Hooks and iterative sprints to help you iterate quickly and ship often.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add astrosteveo/superclaude-marketplace
```

## Available Plugins

### SuperClaude

**Description:** Core development and debugging skills for Claude Code, makes use of Agent Teams, and iterative sprints to iterate quickly and ship often.

**Categories:** Testing, Debugging, Collaboration, Meta, Orchestration
